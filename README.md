# Diabetes and Obesity Prediction using Machine Learning

## Overview
This project predicts the likelihood of Diabetes and Obesity using machine learning classification algorithms based on patient health data.

## Dataset
- diabetes.csv – Medical attributes related to diabetes
- obesity.csv – Health indicators related to obesity

## Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine 

## Tools & Technologies
- Python
- Jupyter Notebook
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Workflow
1. Data loading and preprocessing
2. Feature selection
3. Model training
4. Model evaluation
5. Prediction
